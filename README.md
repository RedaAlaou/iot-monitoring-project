# IoT Monitoring Project

A complete IoT monitoring platform with device management, telemetry, and system performance monitoring.

## Features
- **Device Management**: Manage IoT devices (sensors, actuators) and system devices (computers, servers).
- **Telemetry Monitoring**: Real-time telemetry data visualization (temperature, humidity, CPU, RAM, GPU, etc.).
- **Event Logging**: Track device events (low battery, connection lost, etc.).
- **Frontend Dashboard**: React-based UI for monitoring and managing devices.
